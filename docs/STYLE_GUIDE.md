# Style guide documentation

- Français simple et clair
- Une phrase = une idée
- Expliquer les commandes avant de les montrer
- Toujours documenter les changements visibles pour :
  - installation
  - exécution
  - mise à jour
  - ajout d’un jeu
- Si un script change, vérifier :
  - INSTALLATION.md
  - DOCUMENTATION_TECHNIQUE.md
  - UTILISATEUR.md
  - AJOUT_JEU.md
- L’IA propose, l’humain valide
